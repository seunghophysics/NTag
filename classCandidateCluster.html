Detailed Description

The Cluster of Candidate objects.

This class inherits from

Values can be set and retrieved using Candidate::Set and Candidate::Get functions.

One can pass an unsigned integer index that may be useful in positioning the candidate among hits.

See also
EventNTagManager::FindFeatures for the use of Candidate::Set functions and how the passed hit index in the constructor could be used to position the candidate among PMT hits and fetch relevant features.
